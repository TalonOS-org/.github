## Updates & Maintenance

### Automatic Updates

```bash
# Enable automatic updates
sudo talon-update --enable-auto

# Check for updates
talon-update --check

# Install latest updates
sudo talon-update --install-all
```

### Tool Management

```bash
# List installed tools
talon-tools --list

# Update specific tool
talon-tools --update nmap

# Install new tools
talon-tools --install subfinder
```
