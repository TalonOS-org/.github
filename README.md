- **Storage:** 64GB SSD
- **Network:** Stable Internet connection

### Recommended
- **CPU:** Intel i7 / AMD Ryzen 7 (8+ cores)

- **Storage:** 256GB+ NVMe SSD
- **GPU:** Dedicated graphics (optional)
- **Network:** High-speed Internet (VPN capable)
